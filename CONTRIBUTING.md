# Contributing

## Local development

```bash
python3 -m pip install -e .
python3 -m unittest discover -s tests -v
```

## Code style

- Keep changes small and focused.
- Prefer the existing standard-library-only approach.
- Add tests for behavior changes.

## Pull requests

- Describe the user-facing change.
- Note any new CLI flags, output files, or compatibility impacts.
- Include sample output when relevant.
